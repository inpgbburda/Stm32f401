## Need to install:
    1) Compiler and related tools
       arm-none-eabi
       make
            e.g D:\Programy\ARM_toolchain\GnuWin32\bin
       cmake
            e.g D:\Programy\ARM_toolchain\Cmake\bin\cmake
       mingw - required for Cmake generation 
    2) Open OCD
