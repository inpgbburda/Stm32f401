## Need to install:
    1) Compiler and related tools
       - arm-none-eabi
       - make
       - cmake
       - mingw - required for Cmake generation
       - ruby - required for CMock 
    2) Open OCD

To enable VS Code task execution configure paths in 'settings.json' file.
